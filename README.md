# methodCountE

Function that takes in a string word and returns the number of e's in the word

## Built With

* Visual Studio Code
* HTML
* JavaScript

## Tested With

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites:
* Code Editor of your choice

### To clone the repo to your local repository:

``` bash
git clone https://github.com/Kamogelo98/methodCountE.git
``` 

``` bash
cd methodCountE
```

Open ``` index.html ``` in your preferred browser
